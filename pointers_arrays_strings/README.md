# C - Pointers, arrays and strings

This project covers the basics of pointers, arrays, and strings in C programming.

## Files

| File | Description |
| --- | --- |
| [0-reset_to_98.c](./0-reset_to_98.c) | A function that takes a pointer to an `int` as parameter and updates the value it points to to `98`. |

## Requirements
- Allowed editors: `vi`, `vim`, `emacs`
- All files are compiled on Ubuntu 20.04 LTS using `gcc`
- Code should use the `Betty` style.
